# BMI Calculator Web App

A simple, responsive BMI calculator built with HTML, CSS, and JavaScript. This project helps users calculate their Body Mass Index (BMI) and provides visual feedback for tracking health categories.

## Features

- Calculates BMI from weight and height inputs
- Displays BMI value and category
- Works in the browser with no backend required
- Includes charts and storage utilities for enhanced UI behavior

## Project Structure

- `index.html` — main user interface
- `css/style.css` — styling and responsive layout
- `js/app.js` — app initialization and event handling
- `js/bmi.js` — BMI calculation logic
- `js/charts.js` — chart rendering support
- `js/storage.js` — local storage interactions
- `js/ui.js` — user interface updates

## Usage

1. Open `index.html` in a browser.
2. Enter your weight and height.
3. Click the calculate button to see your BMI and category.
